Welcome to the Cointracker.io Hotbit Preprocessor

This open-source tool was built to work around limitations with cointracker's Hotbit integration. To fix your Hotbit tax reporting and portfolio monitoring, follow the instructions below:



1. Download your complete order history from hotbit.io. Go to Orders > Spot Order > Order (not Order History!) > Export to generate a csv. Rename it to "orders.csv"

2. In a text editor, create a new file called "deposits.txt". In hotbit, navigate to Wallet > Transaction History > Deposits History. For each page, highlight and copy the data from this table and paste it into your file. Paste content from subsequent pages on a new line below the end of the previous page's data. The table column names should be copied from the first page but no subsequent pages. When finished, your file should look like a single long column of data, a single value per row, with the table header names occupying the first 7 rows.

Repeat this process with the Withdrawals History (withdrawals.txt) and System Deposits (system_deposits.txt) pages if they contain any data. Double check that all csv names are correct.

3. Use the form below to upload your .csv and any .txt files that you made. Select multiple files in the popup dialogue with cmd/ctrl + click.
